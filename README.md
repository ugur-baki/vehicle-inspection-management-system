# Vehicle Inspection Station, Appointment, and Payment Management System

This project is a management system developed for vehicle inspection stations.
It handles vehicle inspections, appointment scheduling, and payment processes through a desktop application.

## Technologies Used
- Java
- Java Swing
- PostgreSQL
- JDBC

## Features
- Vehicle inspection record management
- Appointment scheduling system
- Payment processing and tracking
- User-friendly desktop interface using Java Swing
- PostgreSQL database integration
- Data persistence and relational database structure

## Database Setup
This project uses PostgreSQL as its database system.

To run the project locally:
- PostgreSQL must be installed
- A database should be created manually
- Required tables must be created according to the project schema
- Database connection settings must be configured in the application

> Note: The database is not included in this repository and must be set up separately.

## How to Run the Project
1. Clone this repository.
2. Open the project in an IDE such as IntelliJ IDEA or Eclipse.
3. Set up the PostgreSQL database as described in the Database Setup section.
4. Update database connection settings if necessary.
5. Run the application.
